Double Bubble: A Coding Project

Everyone talking at the same time is double trouble.. no.. Double Bubble! This is a javascript coding exercise using loops. We print numbers from 0 to 100 and for multiples of the values provided by the user, we display "Double" for the first value and "Bubble" for the second value. For multiples of both, we display "Double Bubble".

Instructions

Input the values that you want to check against to generate the Double Bubbles. Hit the button and watch the magic happen.
